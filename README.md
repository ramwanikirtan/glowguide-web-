# GlowGuide

An AI-powered skincare and wellness dashboard. Track hydration, build personalized daily routines, and connect with verified dermatologists.

## Features
- **AI Skincare Routines:** Personalized routines based on your skin profile and AI consultations.
- **Water Tracker:** Stay hydrated with daily goals and reminders.
- **Product Library:** Curated product recommendations tailored to you.
- **Find a Dermatologist:** Discover and connect with verified skin specialists near you.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend & Auth: Firebase

## Setup

1. Copy the `.env.example` file to a new `.env` file and fill in your keys (if applicable).
2. Configure your Firebase settings in `public/firebase-config.js` or via your `.env` loader.
3. Serve the application using a local server (e.g., Live Server or `npx serve public`).
